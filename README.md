# MEDIAIRE_TOOLBOX

Shared toolbox for our pipelines. 
* Logging conventions.
* Queue / Daemon classes.
* TransactionsDB (a generic way of persisting state through our pipelines).
* Data cleaner (a tool to periodically clean data on folders).
